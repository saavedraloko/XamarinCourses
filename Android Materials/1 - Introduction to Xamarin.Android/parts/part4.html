Introduction to Xamarin.Android

Duration

5 minutes

Tip: If you are doing this exercise live in a session, make sure to make good use of the instructor, they are online to answer any questions you have!

Goals

The goal of this exercise is to use the Android SDK Manager to install the latest version of the Android SDK on your development machine.

Many of the build tools and APIs you use will either update automatically or notify you when an update is available. That's not the case for the Android SDK. This exercise shows you the manual steps you will need to follow to keep your installation up-to-date.

Required assets

There is no code to write here nor is a Xamarin.Android solution required.

Steps

Below are the step-by-step instructions to implement the exercise.

Update your Android SDK

  1. Launch either Xamarin Studio or Visual Studio.
  2. Open the Android SDK Manager. The image below shows the menu entry you should use in each IDE.
  3. Update the Android SDK Tools, Android SDK Platform-tools, and the Android SDK Build-tools to the latest versions. The image below highlights the three entries you should update. Note that the versions you install may be more recent than the ones shown in the image.
  4. Install the latest version of the SDK Platform. The image below highlights an example of this. Note that the version you install may be more recent than the one shown in the image.

Summary

In this exercise, you updated your Android SDK. This is something that you will need to do periodically and you need to do it manually. At the present time, there is no notification when new versions are available nor is there any auto-update mechanism available.
